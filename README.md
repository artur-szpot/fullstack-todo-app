# fullstack-todo-app

An application to keep track of pending tasks, utilizing Nest.js, React.js, PostgreSQL and Docker

# Features

- Authentication, including logging in through Google
- Ability to create, edit, archive, and delete tasks
- Tasks can be tagged and given a deadline
- Dashboard showing upcoming tasks, sortable by dates, tags and searchable by title or content
- Fully responsive UI for mobile users, dark mode and font size selectors for increased accessibility
